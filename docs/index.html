ara-network-node-dcdn

An Ara Network node that runs a decentralized content distribution node.

Status

Installation

$ npm install ara-network ara-network-node-dcdn

Configuration

ara-runtime-configuration is a dependency of ara-network and will either read from the nearest .ararc. Install ara-runtime-configuration separately to specify default values not present in an .ararc.

Examples

Command Line (ann)

Invoke a network node with the ann (or ara-network-node) command line interface for :

$ ann --t dcdn
